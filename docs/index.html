About me

I’m a CS + Psychology graduate who found programming in 2021 and realized it was something I genuinely enjoyed. It let me reconnect with the part of me that’s always liked working with computers and figuring out how things fit together.


I started with Java and C, which gave me a solid base in problem-solving and understanding how software works. Now I build web applications with JavaScript and Next.js, focusing on improving my skills and learning how to turn ideas into something functional on the screen.


Outside of coding, I like to run, read fiction, spend time with my dog, and make music. Growing up with ADHD, I was often the odd one out, so having hobbies has always helped ground me. Programming has been no different for me. Through my code, I am able to create applications that feel like extensions of myself and in the process, learn a bit more about what I enjoy and who I am.

Pika-Clone Quiz

A personality quiz inspired by the Pokémon Mystery Dungeon games I grew up with as a kid. PMD is Pokémon’s take on the isekai trope, where the main character is pulled from modern life into a new world. The games always begin with a “portal” that asks the player a series of questions to determine which Pokémon they are. This project recreates that portal as a web quiz, with the twist of using a pool of Pikachu-variants to represent different personalities.
