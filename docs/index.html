Pika-Clone Quiz

This is a personality quiz I coded using NextJS. This is my first NextJS project where I got to take advantage of NextJS's ability to create static and dynamic routes. These routes are used to create subsequent pages without having to completely refresh the page each time. I also got a lot of practice using components and tailwind to style my containers. The quiz was implemented using a json file which contains an array of questions where each questions contained an array of answers with each answer contained an array of associated attributes and their weight. The total weight of all the attributes added from each answer will result in the user's final score.
